InstanceMethods は Application に include されるので、 結局 app.initialize の時に呼ばれることになる。

  • run :initialized

  • run :before_configuration
  • (load ‘config.rb’)

  • run :development_config if specified
  • run :instance_available
  • (reload I18n)
  • run :after_configuration
