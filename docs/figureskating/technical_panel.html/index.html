　「3Lz、レビューエッジ」

　「次はジャンプです」

　「3F3T、セカンドレビュー」

　「次はステップです」

　「フライングキャメル、、（バタフライ、、）レベル３」

　「次はステップです」

　「ステップシークエンス」

　（ツイズルからロッカー、、、チョクトウ、、、ループ、、、左のロッカーカウンターツイズル、、、、）

　（、、、、左、、、左、、右右、、、左、、、）

　「ボディなし」

　「ローテーションあり」

　「レベル３」

　「次はスピンです」

　「スピン・コンボ、、、、、足変え、、、、レベル３」

。。。

演技終了：レビュー

　「エレメンツ１２個、レビューは３つです。エレメンツ１番、トリプルルッツ」

　「」

ダメなレビュー例（あくまでフィクションです）

　「エレメンツ３番のトリプルループですが、これは回転足りてますよね」

　「いや、これは足りてません。アンダーです」

　「は？これは十分足りてるでしょ。回りきってる」

　「どこ見てるんですか。全然足りてない。ちょっとスロー頂戴」

　「、、え、、、ちょっと待って、、、」

　「何やってんの早く」

　「あ、、、ハイ、、、（ポチっとな）」

　「ほら足りてる」

　「足りてない」

　「だから足りてるって！足りてないのはアンタの頭でしょっ！」

　「ちょっと何それ！どういうこと！！」

　「、、、あの、、、ユミちゃんアキちゃん、落ち着いて、、、、」

「「うっさいわね！！！」」

（。。。。早く決めてくんないかな。。。）
